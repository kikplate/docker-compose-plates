# n8n

n8n workflow automation platform with PostgreSQL.

## Quick Start

    cp .env.example .env
    docker compose up -d

## Access

* n8n: http://localhost:5678

Log in with the credentials set in N8N_BASIC_AUTH_USER and N8N_BASIC_AUTH_PASSWORD.
Set N8N_ENCRYPTION_KEY to a long random string before creating any credentials.
